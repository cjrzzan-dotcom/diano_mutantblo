이번 압축은 '가방 + 상태창 정리 + 물약 이모지 통일'만 정확히 적용하는 집중 패치입니다.

반영 내용
- 가방 명령어 !가방 추가
- 전투 버튼에 🎒 가방 버튼 추가
- 상태창에서 골드 / 재료 / 속성석 / 부활권 / 강화속성 제거
- 그 정보들을 전부 가방으로 이동
- 물약 버튼 / 전투창 / 상점 표시 이모지를 전부 🧪 로 통일

적용 순서
1. patch_1_status_and_bag.js
2. patch_2_battle_buttons.js
3. patch_3_compact_potion_line.js
4. patch_4_commands_and_interactions.js
5. patch_5_help_and_shop_text.js

적용 후 확인
- !가방 이 동작하는지
- 전투창에 🎒 가방 버튼이 보이는지
- 상태창에 골드/재료/속성석/부활권이 사라졌는지
- 물약 버튼 이모지가 전부 같은지
